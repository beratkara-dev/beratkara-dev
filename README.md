# Hi there, I'm Berat 👋
### aka "Arround" 

🚀 **Full-Stack Developer | Security Researcher | Hardware Enthusiast**

I'm a 16-year-old developer based in Kocaeli, TR. I thrive at the intersection of low-level system mechanics and modern software architecture. Whether it's reverse-engineering game engines, building asynchronous desktop tools, or designing full-stack e-commerce platforms, I love tearing things apart to understand how they work—and then rebuilding them better.

### 🎯 What I Do
* 🛡️ **Game Security & Reverse Engineering:** Exploring client-server architectures, designing obfuscation algorithms, and developing anti-cheat solutions (Lua/Roblox, Java/Minecraft).
* 💻 **Desktop & Tooling:** Building custom game launchers, asynchronous management tools, and open-source AI assistants (C#, WPF, .NET).
* 🌐 **Full-Stack Web Architecture:** Founder of [Truva Development](https://truvadevelopment.com.tr), architecting scalable marketplaces and integrating secure payment gateways.
* 🤖 **Hardware & Robotics:** Bridging code and physical world through sensor-driven radar systems and autonomous vehicles (Arduino, C++).
